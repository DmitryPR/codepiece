# mini-algorithms

Tiny TypeScript modules used as a **local scan target** for CodePiece. See **[`../README.md`](../README.md)** for how to aim **`TARGET_REPO`** here or at a clone of [Lugriz/typescript-algorithms](https://github.com/Lugriz/typescript-algorithms).

Modules: **binary-search**, **fibonacci**, **hamming-distance**, **queue** (class), **gcd** / **lcm**, **clamp**, **strings** (reverse / palindrome), **sorting** (insertion sort), **stack** (class), **memo** (memoize / once), **arrays** (zip / sum).
